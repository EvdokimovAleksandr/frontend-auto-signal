# Frontend Auto Signal

React приложение с Redux Saga для проекта Auto Signal.

## Быстрый старт

```bash
# Установка зависимостей
npm install

# Запуск dev сервера
npm run dev

# Сборка для production
npm run build
```

## Структура проекта

```
src/
  components/     # Переиспользуемые компоненты
  pages/          # Страницы приложения
  store/           # Redux store (slices, sagas)
  services/        # API сервисы
  config/          # Конфигурация
  types/           # TypeScript типы
```

## Технологии

- React
- Redux Toolkit
- Redux Saga
- TypeScript
- Vite
